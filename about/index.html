我是谁

一个在北京求学的陕西西安人，就读于北京大学医学部公共卫生学院。平时喜欢研究自己感兴趣的事物，真正沉迷的时候，甚至会耽误自己的大学学业，还好 4 年过去，暂无一门挂科。喜爱编程和统计。在 2017 年末把博客从 hexo 迁移到了 hugo 框架之下。

我的特点

描述一个人可以通过许多维度，但让人印象深刻的总是那些最鲜明的特点。

我热爱阅读和电影。2017 年阅读了 73 本书，最喜欢的电影是大卫芬奇的《社交网络》。所有读过的书和看过的电影都在豆瓣上有标注。关于阅读和电影，我有非常严重的松鼠症，在硬盘里囤积了无数的书籍和电影，还没有来得及去看，这可能也是我无法改正的毛病。

我热爱设计和美。设计包括软件的 UI 设计，海报设计，电子产品的工业设计，文档的排版设计等等等等。我会因为美而容忍其他的一些缺点。再比如我几乎是我的同学中唯一率先离开 Word，尽力使用 Markdown 和 LaTeX，已经多次使用 LaTeX 完成作业、实验报告和论文综述等。

我的目标

  • 在我热爱的统计领域持续阅读学习，并通过博客的方式记录我自己的成长

  • 宁愿使用碎片化的时间阅读收藏的博客，而非刷微信和朋友圈
